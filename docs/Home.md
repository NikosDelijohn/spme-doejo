SPME-DoEjo is a prototypical web-based tool designed to optimize Solid-Phase Microextraction (SPME) parameters using a [Box-Behnken Design (BBD)](https://en.wikipedia.org/wiki/Boxâ€“Behnken_design). It allows users to input CAS numbers, select the correct compounds, specify experimental properties, and generate optimized SPME conditions.

This tool is intended for researchers, chemists, and laboratory technicians working on trace analysis and environmental or analytical chemistry projects. 

In this wiki, you can find detailed information on:

- How the tool computes basic SPME parameters before performing a BBD DoE.
- Guidance on selecting appropriate compounds and fibers.
- References to relevant literature and methodologies.

Whether you are new to SPME or experienced with design of experiments (DoE), this wiki will help you get the most out of SPME-DoEjo.
